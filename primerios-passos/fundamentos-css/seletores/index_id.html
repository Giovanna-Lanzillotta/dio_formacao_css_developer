Bem Vindo!

No CSS nós possuimos os seletores por ID

Olá tudo bem?

  • Seletor por Tipo:Busca elementos por uma tag HTML específica
  • Seletor por ID(#):Busca elementos através do atributo "id"
  • Seletor por Classe(.):Busca elementos atráves do atributo "class"
  • Seletor Universal(*):Seleciona todos os elementos HTML
  • Seletot de atributo:Seleciona elementos que possuem um atributo específico em nosso documento HTML e conseguimos também buscar atributos com um valor específico.
