Cores

Existem várias formas definirmos cores utilizando o CSS.Elas são:

  • Cores pré-definidas
  • Palavra-chave "currentcolor
  • Hexadecimal
  • Hexadecimal com transparência
  • RGB
  • RGBA
  • HSL
  • HSLA

HSL

hsl (Hue, saturation, lightness)
  • Hue (matriz): é o grau na roda de cores (de 0 a 360 graus)
    • 0 ou 360 é vermelho
    • 120 é verde
    • 240 é azul
  • Saturation(saturaçaõ): valor percentual
    • 0% um tom cinza
    • 100% cor total
  • Lightness:
    • luminosidade da cor
    • 0% é preto
    • 100% é branco
